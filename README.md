# Gimbal CLI

Gimbal is a cross-platform developer environment bootstrapper built in Java.

### Features
- Install tools (Java, Maven, Git, Docker)
- Command validation (`install java maven --help`)
- Flag system (`--help`, `--all`)
- Error handling with structured exit codes
- Human-friendly logger system

### Usage

```bash
gimbal install java
gimbal install java maven --help
gimbal --help
